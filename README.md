# Supermarket-Sales-Analysis-Python-Power-BI-Dashboard

📌 Project Overview
This project focuses on analyzing supermarket sales data to uncover key business insights regarding store performance and customer behavior. The workflow involves Data Cleaning using Python and creating an Interactive Dashboard in Power BI for data-driven decision-making.

📊 Dashboard Key Features
Total Revenue: Successfully tracked a total revenue of $118.58K.

Gender Analysis: Analyzed sales distribution between Male ($64K) and Female ($54K) customers.

Branch Performance: Monitored real-time performance across three main branches: Mandalay, Naypyitaw, and Yangon.

Product Category Trends: Visualized top-performing categories like Electronic Accessories and Food & Beverages.

Payment Insights: Identified customer preferences for Cash, E-wallet, and Credit Card payments.

🛠️ Tech Stack
Python (Pandas): Used for Data Cleaning and Exploratory Data Analysis (EDA).

Power BI Desktop: Used for building interactive visualizations and the final dashboard.

DAX (Data Analysis Expressions): Used to create custom measures for Revenue and Sales totals.

Theme: Implemented a professional CY26SU02.json theme for a modern look.

📁 Repository Structure
Analysis_Code.ipynb: Python script for data processing.

Supermarket_Dashboard.pbix: Main Power BI dashboard file.

supermarket_sales.csv: Raw dataset used for this analysis.

🚀 How to Run the Project
Python: Open the .ipynb file in Jupyter Notebook to view the data cleaning steps.

Power BI: Open the .pbix file in Power BI Desktop to explore the interactive visuals.
